This project is the "source code" of the portal: http://www.semantictext.info
You can use it to build your own portal or make your templates.

Use as you like (respecting the licence, of course).

REQUIREMENTS:

	JDK: Check with command line: java -version
	Ant: Check with command line: ant -version

EXECUTION:

	Execute with command line: ant build

	Thats all!! 
	You should see a new directory //web-out// with the portal.

EXPLANATION:
	This project is so easy ;-)
	Change build.xml as you like.
